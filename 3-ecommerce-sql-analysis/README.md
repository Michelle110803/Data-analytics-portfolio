# E-Commerce SQL Analysis 

## Overview 
Recreated the Excel e-commerce analysis using SQL in DB Browser for SQLite

## Queries Included 
- Sales by Category
- Most Returned Products
- Orders by Status
- Monthly Sales Trend
- Order Value Segmentation

## Key Findings 
- Books is the top category with $38,357 in sales
- Fiction is the most returned product
- 34 out of 73 orders are High Value (above $1,500)
- Sales peaked in February 2025

## Tools Used 
- DB Browser for SQLite
- SQL (SQLite dialect)
